# Savage Planet GCS Library

Official GURPS Character Sheet files for the Savage Planet tabletop role-playing setting.

## Player downloads

Open the appropriate folder, select the file, and use **Download raw file**. Save the file locally, then open or import it with GURPS Character Sheet.

## Repository layout

- `characters/` — complete `.gcs` character files
- `equipment/weapons/` — individual weapon `.eqp` files
- `equipment/armor/` — armor `.eqp` files
- `equipment/ammunition/` — ammunition `.eqp` files
- `equipment/vehicles/` — vehicle-related equipment files
- `equipment/general/` — tools, supplies, and miscellaneous gear
- `libraries/` — consolidated GCS libraries
- `templates/` — reusable character templates
- `traits/` — custom trait libraries
- `skills/` — custom skill libraries
- `modifiers/` — advantage and equipment modifier libraries
- `documentation/` — campaign rules, change notes, and usage instructions

## Canon and revisions

Only files committed to the `main` branch are considered approved Savage Planet game material. Stable filenames are retained when entries are revised so player download links continue to work.

## GCS compatibility

Files are produced for GURPS Character Sheet 5.44.x unless a file or release note states otherwise.

## Campaign rule note

Savage Planet does not use the GURPS After the End Freakishness subsystem. Mutations are represented directly as advantages, disadvantages, features, attacks, or custom traits.
