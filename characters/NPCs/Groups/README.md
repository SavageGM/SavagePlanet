# Groups

NPCs organized by faction, organization, settlement, crew, military force, gang, or other defined affiliation.

Each group receives its own subdirectory:

`characters/NPCs/Groups/<Group Name>/`

Store associated individual character files inside that group directory rather than in `characters/NPCs/Individuals/`.

Example:

`characters/NPCs/Groups/Empire/`

A fully developed Imperial character belongs in the Empire directory because affiliation takes priority over the general Individuals category.
