# Bestiary

This directory contains GCS files for nonhuman living and quasi-living entities in Savage Planet, including:

- Animals
- Mutant animals
- Creatures
- Monsters
- Plants
- Mutant plants
- Fungi
- Organisms
- Other bestiary entries

Sentient faction members belong under `characters/NPCs/Groups/` when they are part of an organization. Robots belong under `characters/NPCs/Robots/` unless they are better treated as a creature entry for gameplay purposes.
