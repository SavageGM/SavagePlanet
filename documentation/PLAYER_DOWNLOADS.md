# Player Download Instructions

1. Open the desired file in this repository.
2. Select **Download raw file**.
3. Save the file without changing its extension.
4. Open GURPS Character Sheet.
5. Open a `.gcs` character directly, or import/open the appropriate GCS library file such as `.eqp`, `.adq`, `.skl`, `.adm`, or `.eqm`.

## Compatibility

Savage Planet files are currently intended for GURPS Character Sheet 5.44.x.

## Stable links

Approved files remain at stable paths on the `main` branch. Updated versions replace the file at the same path so bookmarks and player download links continue to work.

## Canon status

Files on `main` are approved for use. Work-in-progress files should be kept off `main` until reviewed.
